# Probabilitat

## Espai mostral

**Def**: conjunt de ressultats possibles d'un experiment.

[Succes/esdeveniment][esdeveniment]: $A\subset E$

- Segur => $E$
- Imposible =>$\emptyset$

### Propietats

1. reset
   1. $A\cap E=E$
   2. $A\cup\emptyset=\emptyset$
2. invariant
   1. $A\cup E=A$
   2. $A\cap\emptyset=A$
3. Assosiativa
   1. $(A\cup B)\cup C=A\cup (B\cup C)$
   2. $(A\cap B)\cap C=A\cap (B\cap C)$
4. Distributiva
5. Commutativa
